# Rexel SKU Reference

## SKU Format

Rexel SKUs are alphanumeric codes from their product catalog. Common patterns:

| Pattern | Example | Meaning |
|---------|---------|---------|
| `RMX___` | RMX142WG250CL | Romex wire (gauge-conductor-length) |
| `DLR___` | DLR46071203KWH | Downlight/recessed lighting |
| `LB___` | LB115-DFT | Panel/breaker related |
| Number-only | 30-1345J | Rexel item number |

## Search Tips

- **Wire:** Search "romex", "thhn", "wire", "cable", "RMX"
- **Breakers:** Search "breaker", "BR", "QO", "GFCI"
- **Lighting:** Search "LED", "downlight", "DLR", "troffer", "fixture"
- **Switches:** Search "switch", "dimmer", "decora", "toggle"
- **Receptacles:** Search "recept", "outlet", "GFCI", "duplex"
- **Conduit:** Search "EMT", "PVC", "conduit", "coupling", "LB"
- **Boxes:** Search "box", "cover", "ring", "gang"

## Units

| Code | Meaning |
|------|---------|
| EA | Each (individual items) |
| FT | Feet (wire, cable) |
| CL | Coil (wire sold by coil, often 250ft) |
| BX | Box (bulk packaging) |
| RL | Roll |

## Price Notes

- Unit prices are per-unit (per foot for wire, per each for devices)
- Wire is often quoted per-foot but sold in 250ft coils
- Prices include any negotiated contract discounts with Rexel
- Price variance shows fluctuation over the invoice history period
